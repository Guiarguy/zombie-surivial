Version 1.0  
There are four kinds of zombies and two scenes.  
All of zombies are spawn by instantiate in scene.  
There are three rounds, if player persist alive through three round then player win.  
The zombie spawn speed and property will gain after each round.  

# How to use
1. clone the game into your mechine
  code : ```git clone https://github.com/tmpaulhsun/zombie-surivial-unity-3d-game.git```
2. download unity editor to run the project
3. find the add button in unity editor and open the folder you clone
4. start to play!
